Keywords: hypothesis testing, event funnel, digital marketing, user journey

In this project we're assessing the A/A/B test running in an online food store. Out of three groups of users, one group got a website version with altered fonts. We're going to find out if it had an impact on their conversion rate.

This is a study project for Yandex.practicum Data Analyst program. Please feel free to open issues or PR if you have any comments!
